## 高性能模式

如果电脑电源模式设置中没有高性能模式，可通过以下的操作开启高性能模式，以便 GPU 及 CPU 能够高功率工作

**高性能模式：**

> Win+R 键搜索 cmd，打开 cmd 输入以下代码开启

```cmd
powercfg -SETACTIVE 8c5e7fda-e8bf-4a96-9a85-a6e23a8c635c
```

浏览器搜索可加如下后缀进行限制:

> site:xxx.com //通过该限制可使搜索显示优先为 xxx.com，可以便于检索官方信息用
> filetype:ppt/pdf... //通过该限制可使搜索显示游戏为 ppt/pdf...限定格式，便于资料收集
